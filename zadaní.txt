✅ Nejlepší základ pro tvoje zadání
ytmimi/Chess.com-Public-Game-Data-API
Program stahuje data přímo z Chess.com public API a pro každou hru sbírá: username, rating, národnost, barvu (bílá/černá), výsledek (výhra/prohra), první tah, varianta, počet tahů, datum. Výstupem je soubor chess_data.csv. GitHub
➡️ Toto je tvůj sběr dat — reálná data, z API, s doložitelným zdrojem. ✔️

🧠 Co na to nastavit jako ML model
Ideální klasifikační model: "Předpověď výsledku partie" (výhra / prohra / remíza)
Atributy (splňuje požadavek 5+ atributů):

Rating hráče
Barva (bílá / černá)
Počet tahů
První tah (opening)
Časová kontrola (bullet / blitz / rapid)
Výsledek → label pro klasifikaci

1500+ záznamů? Pokud máš na chess.com 500+ her, stáhneš data i od jiných hráčů přes API — splnitelné snadno.
